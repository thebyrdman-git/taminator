## The Tools

Taminator includes 4 core automation tools:

### 1. `tam-generate-agenda` (Replaces KAB)
Intelligent TAM call agendas with:
- Cross-case pattern detection
- Trend analysis
- Priority scoring
- Account health metrics
- Proactive recommendations

### 2. `tam-backlog-cleanup` (Replaces kab-backlog)
Smart backlog management with:
- Auto-close detection

### 3. `tam-t3-reader` (Replaces t3)
T3 intelligence with:
- Product-based filtering

- Recommendation engine

### 4. `tam-coverage` (Replaces kab-coverage)
Coverage automation with:
- Professional announcements
- Comprehensive backup briefings
- Customer context integration
- Strategic handoff preparation
