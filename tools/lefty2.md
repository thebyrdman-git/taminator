# lefty2

Red Hat internal case reporting and analytics tool.

## Location
`lefty2` (system command)

## Description
Command-line tool for generating case reports, analytics, and summaries from Red Hat support data. Provides TAMs with reporting capabilities beyond basic case lookups.

## Key Features
- Case reporting and analytics
- Summary generation
- Trend analysis
- Account-level reporting
- Integration with Red Hat support systems

## Common Use Cases
- Generate weekly/monthly case reports
- Analyze case trends by account
- Create executive summaries
- Track case metrics

## Integration Points
- Complements rhcase for reporting
- Used for weekly TAM reports
- Feeds into executive briefings
- Source for trend analysis

## Notes
- Actively used tool (replaced lefty-2-cli project)
- Requires VPN/internal network access
- Output formats suitable for further processing
