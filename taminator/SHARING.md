# Sharing Taminator with Coworkers

**Official repo (only path for now):** https://gitlab.cee.redhat.com/jbyrd/taminator  
**Project title:** TAMINATOR - RFE and Bug Report Generator  
**Project description (use on GitLab and elsewhere):** A tool for generating and maintaining RFE and bug reports for TAMs.

Use this repo only when sharing with Red Hat coworkers. Do not use GitHub or other remotes as the source of truth.

- **Clone (recommended path on Mac and Linux):** `git clone https://gitlab.cee.redhat.com/jbyrd/taminator.git ~/taminator` — then run from `~/taminator/taminator`.
- **Releases / downloads:** https://gitlab.cee.redhat.com/jbyrd/taminator/-/releases (latest: [v2.1.6](https://gitlab.cee.redhat.com/jbyrd/taminator/-/releases/v2.1.6); tree: `releases/v2.1.6/` for AppImage, DMG)
- **Issues / feedback:** https://gitlab.cee.redhat.com/jbyrd/taminator/-/issues

Requires Red Hat VPN and GitLab CEE access. Point coworkers to this repo and to the main [README](README.md) for setup and usage.
