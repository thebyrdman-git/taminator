### Installation

#### Windows
1. Download `Taminator-Setup-1.9.2.exe`
2. Run the installer
3. Choose installation directory
4. Launch from Start Menu or Desktop shortcut

#### macOS
1. Download the appropriate DMG for your Mac
2. Open the DMG file
3. Drag Taminator to Applications folder
4. Right-click and select "Open" (first time only for unsigned apps)

#### Linux
```bash
chmod +x Taminator-1.9.2.AppImage
./Taminator-1.9.2.AppImage
```
