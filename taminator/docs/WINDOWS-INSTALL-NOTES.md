# Windows install notes

- **Version in the app:** The UI shows the version from the build (e.g. "Taminator v2.1.3"). If you see an older version than the installer you ran, uninstall Taminator first (Windows Settings → Apps → Taminator → Uninstall), then run the new installer.
- **Where to get the installer:** [GitLab releases](https://gitlab.cee.redhat.com/jbyrd/taminator/-/releases) — download the Windows `.exe` for the version you want. Red Hat VPN and GitLab CEE access required.
